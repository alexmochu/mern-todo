# MEARN-todo
The MERN stack consists of MongoDB, Express, React / Redux, and Node.js.

# Install
1. Clone the repo: 
```
$ git clone [URL]
```
2. Install dependencies.
```
$ yarn install
```
3. Run the app:
```
$ yarn dev
```
